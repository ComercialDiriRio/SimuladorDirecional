# Front-end web (HTML / CSS / JS)

Ficheiros estáticos servidos por FastAPI em `/static/...` quando se usa [`simulador_dv/api/main.py`](../simulador_dv/api/main.py).

- `index.html` — página de demonstração (health, métricas Pro Soluto, login demo).
- `css/app.css` — tema com variáveis `:root` alinhadas ao design anterior (sem seletores do DOM Streamlit).
- `js/app.js` — chamadas `fetch` à API (`/api/*`).

Arranque: ver secção **UI web** em [`simulador_dv/README.md`](../simulador_dv/README.md).
